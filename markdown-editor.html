About Inkwell

Version 1.2

Inkwell is a browser-based Markdown editor with a live side-by-side preview. It supports formatting tools, autosave, import and export, find and replace, an outline, syntax-highlighted code, Mermaid diagrams, KaTeX math, sortable tables, task lists, and custom roadmap blocks.
